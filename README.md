### Installation

```bash
# clone the repository on your computer
$ git clone git@github.com:bmazurme/.....git

# install dependencies
$ npm install

# run dev mode
$ npm run dev

# build project
$ npm run build

# clear project
$ npm run clear

# run eslint
$ npm run eslint

# launch
$ npm start
```
